Wikipendiums are written using markdown syntax.
The full specification is available, if you're into that.
This is the short version:

Heading

Headings are defined by using #. The largest heading is # Heading 1, then comes ## Heading 2 and so on.

Paragraphs

Paragraphs are separated with a blank line.

So this line, and the next line.
Is actually the same paragraph.

While this becomes a new one.

Bold

__These two underscores make the text bold.__

Italic

_One underscore, and the text becomes italic._

Blockquotes

> in front of the text gives a blockquote.

> You can
even split the text to separate lines
and it will still be the same blockquote.

Unordered lists

- This
- is
- a
- list
    - And you can have sub-lists too
    - by indenting them with a tab

Ordered lists

1. While this one...
2. has numbers.
1. And you don't have to know how to count.
5. We will correct the numbers for you.

    23. number the subsections
    1. properly
6. and continue with the right numbers here

Definition lists

Apple
:   Something you can eat

Car
:   Something you cannot eat

Links

[This is a link](http://tothispage.com/)

If you want to show the link URL, you can simply type:

<http://theurl.com/>

To link to other sections in the article, use this format:

[Section Name](#section-name)

Images

![Alt text](http://url.to/image.png)

This table markup should be quite self-explanatory.
Unfortunately it's not well-suited for advanced table layouts, so try to keep it simple.

|| __Heading 1__ || __Heading 2__ ||
|| Row 1, Col 1  || Row 1, Col 2  ||
|| Row 2, Col 1  || Row 2, Col 2  ||

Mathematical notation may be written as follows:

Inline math uses single dollar signs, like this $ y = x $, while blocks of math are indicated with double dollar signs:

$$ y = 2*x $$

We support standard LaTeX syntax

$$ e^{ \pm i\theta } = \cos \theta \pm i\sin \theta $$

This piece of code will show Euler's formula:

$$ e^{ \pm i\theta } = \cos \theta \pm i\sin \theta $$

With LaTeX you can type out formulas, matrices etc. See a full LaTeX/Math documentation here.

Margin Notes

[#] This is a margin note that can provide additional
information or commentary to the main text of a compendium.

Blocks of code

    Indent lines with four spaces (or a tab) to indicate code snippets.
    They will render just like what you are reading now (meta!).
